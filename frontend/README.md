# Frontend (NextJS)

## Prerequisites
- Node.js v23.5.0
- npm v11.0.0

## Setup

### Restore Dependencies
To install the required dependencies, run:
```bash
npm install
```

## Run Component Tests
Run following command:
```sh
npm run test
```

### Run in Development Mode
To start the application in development mode, run:
```bash
npm run dev
```
